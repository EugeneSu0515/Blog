[Flutter]初遇Flutter

9月 03, 2018 分類 Flutter

前提情要

筆者在前幾天，被同事問到是否知道Flutter這個技術，當時毫無頭緒。查了一下網路才知道，是由Google發出，使用Dart語言撰寫，經由Flutter編譯後可以生成原生Android及IOS程式，因筆者之前有經驗撰寫Xamarin.Forms，很好奇這種寫一套即可產生App兩大生態系統的原生程式，跟著筆者一探究竟吧。

內容

目錄

  1. 前提情要
  2. 內容
    1. 安裝Flutter環境
    2. 安裝Android Emulator
    3. 撰寫Flutter程式
  3. 結論

安裝Flutter環境

  • 下載Flutter安裝包，筆者這邊以Windows 10系統做示範，進到相對應系統頁面後，找到flutter_windows_v0.6.0-beta.zip點擊位置，將壓縮檔案下載下來。

  • 將下載下來的壓縮檔案解壓縮至C槽特定位置，筆者這邊以官網建議之C:\src\flutter

    此位置到時需設置為使用者環境變數

  • 第二步驟解壓縮檔案中找到bin\flutter.bat並點擊執行

  • 接下來設定使用者環境變數後就完成了

    • 筆者這邊使用Rapid Environment Editor，記得使用系統管理者身份開啟該軟體，在User Variables的Path中新增解壓縮的位置，以筆者環境為例則設定C:\src\flutter\bin
    • 或者使用Windows 10內建之系統>系統內容>環境變數>使用者變數>Path

  • 於cmd中執行看看flutter相關指令

如果出現以上畫面，恭喜你已成功設定Flutter環境。

安裝Android Emulator

可以參考13 best android emulators of 2018，筆者直接以Android Studio做示範。

  • 下載Android Studio
    • Android Studio下載連結 https://developer.android.com/studio/install
    • 或者choco install androidstudio -y

  • 安裝完成後設定Android Emulator

    安裝Android Studio目的不為別的，只為了Android Virtual Device Manager，但筆者始終在以上頁面找不到開啟Android Virtual Device Manager地方，筆者使用最後一個選項Import an Android code sample，隨意選擇一個範例專案並開啟，開啟範例專案後設定Virtual Device

    由以上畫面可以看到，筆者已經設置一台Nexus 6 API 28的裝置，做到這邊已經完成設定Android Emulator的步驟了。

    筆者還是強烈建議透過安裝Android Studio設定Android Emulator，以避免不必要的Android其他環境問題，例：Android SDK

撰寫Flutter程式

官網建議編輯器有兩個，Android Studio 以及 Visual Studio Code，筆者因撰寫過Vuejs程式，兩者比較之下對於VSCode更為熟悉一點，選擇VSCode作為主要開發工具。

  • 安裝VSCode
    • https://code.visualstudio.com/download
    • choco install vscode -y
  • 開啟VSCode並安裝外掛套件
    • Dart
    • Flutter
  • 新增一個hello_world專案
    • Ctrl+Shift+P開啟VSCode Command小視窗
    • 鍵入flutter會有以下三個選擇
      • Flutter: New Project
      • Flutter: Run Flutter Upgrade
      • Flutter: Run Flutter Doctor
    • 選擇Flutter: New Project，鍵入專案名稱：hello_world，最後選擇專案存放位置
  • 專案目錄結構 程式主要撰寫於lib>main.dart中
  • 執行專案前需先啟動Android Emulator，
    • Ctrl+Shift+P開啟VSCode Command小視窗
    • 鍵入flutter會出現更多選項，選擇Flutter: Launch Emulator，會跳出本機已設定之Virtual Device List，以筆者環境來說是Nexus 6 API 28

  • 鍵入Ctrl+Shift+D切換至Debug頁籤，新增Debug Configuration

    使用Add Configuration方式VSCode會自動新增launch.json檔案

  • 最後按下F5偵錯並執行程式，可以看到程式執行於Android Emulator中

結論

這一篇就先以設定環境，執行第一支hello_world專案程式，下一篇開始慢慢講述專案結構、dart語言以及flutter提供的Widgets，下篇見。
