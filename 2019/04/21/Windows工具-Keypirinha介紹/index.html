[Windows工具]Keypirinha介紹

4月 21, 2019

前提情要

筆者使用Mac系統時，非常地依賴Spotlight功能，當時選擇使用擴充功能更齊全的Alfred，主要是純使用鍵盤就能完成所有的事情，是筆者這種寫程式的人追求的境界阿。

換到Windows10的環境時，會想要有類似的工具可使用，筆者起初使用的軟體為Wox，但是常遇到當掉需要重開的狀況，查找了一下終於遇到讓我愛不釋手的工具，就是這個文章的主角keypirinha。

內容

安裝方式

筆者這邊使用choco install的方式

使用方式

按下ctrl+win+k，就會出現keypirinha。

開機自動執行設定

再來設定開機時自動執行，不要再用把建立捷徑，並將捷徑拉近啟動資料夾中，筆者直接使用keypirinha開放的configure的方式設定。

  • 開啟keypirinha主視窗後，按下Configure，選擇Keypirinha:Configure
  • 會開啟兩個文字檔案，名為User的是使用者自行設定，設定方式為ini檔案格式
  • 設定section為app，將launch_at_startup屬性設定為yes，存檔即可，會在下次開機時生效

Package Control

其實跟Wox一樣，Keypirinha開放許多第三方套件，安裝方式官方建議使用 keypirinha package control。安裝方式則README.md所寫，需使用到kepirinha的console功能。

按下ctrl+win+k跳出keypirinha視窗後，輸入關鍵字：console搜尋，選擇Keypirinha:console功能項目

出現Keypirinha專屬打指令的視窗後鍵入

推薦第三方套件

基本上安裝好keypirinha後，已經有預設的套件，如下：

  • Bookmarks
  • Calc

  • Everything

    • 需要自己爾外安裝Everything 軟體

  • FileBrowser

  • FilesCatalog
  • FileZilla
  • GoogleTranslate
  • PuTTY
  • TaskSwitcher
  • URL
  • WebSearch
  • WebSuggest
  • WinSCP

筆者自己需求，另安裝的如下：

  • VolumeControl
    該套件則因人而異，筆者環境是為省桌面空間，筆電會合起來並立於直立架中，接上外接螢幕使用。且現在最上手的FILCO Majestouch 2 NINJA 87鍵紅軸，沒有音量調整的按鍵阿，用滑鼠去調整，實在是懶，有了該套件，我可以用鍵盤快捷鍵的方式調整音量，救星阿。

    • 只要在打開keypirinha主視窗後按下vol就可以調整音量了
    • 你只要按下想設定的音量數字，選項中會出現SetVolumeTo{你設定的音量數字}

  • WindowsApps
    Keypirinha預設無法搜尋到使用「Windows市集」下載安裝的軟體，裝好該套件後才會被搜尋到，滿實用的一個套件。

  • SystemCommands
    關機、重新啟動等系統操作命令，裝好該套件後，於keypirinha上完成，對筆者來說也滿實用的。完整指令則參考。
  • UrlShortener
    這套件有點可惜，因google短網址服務已經終止參考，無法繼續使用了。

結論

筆者每次重灌系統後，一定會先安裝choco跟keypirinha阿，實在是太愛使用他們了，為筆者日常使用電腦帶來方便，在此推薦給大家。
